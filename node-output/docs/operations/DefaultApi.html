This is the API

addlicense

This function adds a new product license to the given account.

URL

https://virtserver.swaggerhub.com/ashleykinard/addlicense/1.0.0/addlicense

HTTP Method

GET

Response Type

inline_response_200

Parameters

  • Query: licenseID String

    The license ID to add to the account.

  • Query: username String

    The account to assign to the license.

  • Query: licenseType String

    The type of license to add to the account: * `personal` - A license for personal accounts. * `corporate` - A corporate account license. * `trial` - A trial license.

  • Query: notes String

    Internal notes about the license activation.
